PharmaTrack Sales Analytics
Tools: PostgreSQL | Power BI | Excel | Kaggle
Description:
Cleaned and analyzed a real-world pharmacy Point-of-Sale dataset from Kaggle, enriched with financial metrics to build a complete pharmacy sales analytics solution.
Key Highlights:
•	Cleaned and standardized a real Kaggle dataset — reduced 300+ inconsistent dosage forms to 30 clean categories using Excel Power Query
•	Designed 7 relational tables enriched with price, revenue, cost, profit, supplier, and employee data
•	Wrote 30 advanced SQL queries including CTEs, Window Functions (Moving Average, MoM Growth, Category Rankings), and 4 Views
•	Built a 5-page Power BI dashboard analyzing 5,000 transactions across 1,357 medicines, 12 categories, and 15 supplier countries
•	Implemented inventory alert system identifying low-stock and expiring medicines using SQL Views
•	Produced Business Analyst reports with actionable pharmacy management recommendations
